Documentation rapide

Synthese des routes, des commandes et du flux de verification.

Demarrage

npm install
npm run seed:all
npm run dev

Routes publiques

GET / POST /users POST /auth/login

Routes protegees

GET /users GET /catways POST /catways GET /reservations POST /reservations PUT /reservations/:id

Tests automatiques

npm run test:api:phase7
npm run test:api:phase7:with-seed

Points d'attention

  • Configurer un vrai JWT_SECRET avant livraison.
  • Verifier la whitelist IP MongoDB Atlas si la connexion echoue.
  • Utiliser le token Bearer sur toutes les pages protegee.
