API Port Russel

Bienvenue sur l'interface d'administration

Cette application permet de piloter les utilisateurs, les catways et les reservations du port depuis une interface web connectee a MongoDB.

Voir la documentation API

Fonctionnalites

  • Authentification JWT avec redirection vers le tableau de bord
  • CRUD utilisateurs, catways et reservations
  • Consultation des reservations en cours
  • Acces rapide a la documentation et aux pages de gestion

Connexion

Une connexion reussie vous redirige automatiquement vers le tableau de bord.
